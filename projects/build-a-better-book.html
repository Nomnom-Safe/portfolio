Build a Better Book

An honors project pairing tactile 3D objects with a children’s book so blind and visually impaired readers could explore the story through touch as well as braille text.

Motivation

Picture books rely heavily on visual narrative cues. This project asked how physical objects could carry some of that narrative weight for readers who experience stories primarily through touch and braille.

What I delivered

  • Sourced 3D files for eight story objects and evaluated prints for durability and tactile recognition.
  • Learned 3D modeling software to improve four existing meshes so they would feel distinct and survive classroom handling.
  • Partnered with Clovernook Center for the Blind and Visually Impaired to donate finished objects and digital files; copies of the companion book in braille traveled with the tactile kit to a school in Africa.
  • Captured supporting web materials in the repository’s Website folder for documentation and lightweight sharing alongside the printable/STL assets.

Artifacts in the repo

The GitHub project bundles STL models, print-ready PDFs, and a simple static website directory for presentation. Treat the repository as the canonical archive for both physical and digital deliverables.
